# Data Engineering Zoomcamp

This repository documents my progress through the Data Engineering Zoomcamp by DataTalks.Club, led by Alexey Grigorev.

The goal of this repository is to track my learning journey, exercises, and practical implementations across the main topics of modern data engineering, including containerization, workflow orchestration, data ingestion, data warehousing, analytics engineering, batch processing, and streaming. It includes Docker, Terraform, Kestra, BigQuery, dbt, Bruin, Spark, Kafka, and other core data engineering tools.

## Course Content

### Module 1: Containerization and Infrastructure as Code #(DONE)

- Introduction to Google Cloud Platform (GCP)
- Docker and Docker Compose
- Running PostgreSQL with Docker
- Infrastructure setup with Terraform

### Module 2: Workflow Orchestration #(DONE)

- Introduction to data lakes and workflow orchestration
- Building workflows with Kestra

### Workshop 1: Data Ingestion

- Reading data from APIs
- Building scalable data ingestion pipelines
- Data normalization
- Incremental loading

### Module 3: Data Warehousing

- Introduction to BigQuery
- Partitioning, clustering, and performance best practices
- Machine learning in BigQuery

### Module 4: Analytics Engineering

- Analytics engineering and data modeling
- dbt (data build tool) with DuckDB and BigQuery
- Testing, documentation, and deployment

### Module 5: Data Platforms

- Building end-to-end data pipelines with Bruin
- Data ingestion, transformation, and data quality
- Cloud deployment with BigQuery

### Module 6: Batch Processing

- Introduction to Apache Spark
- DataFrames and SQL
- Internals of GroupBy and Joins

### Module 7: Streaming

- Introduction to Kafka
- Kafka Streams and KSQL
- Schema management with Avro
